This project implements a dynamic environmental tracking solution using IoT sensors mounted on movable vehicles. Unlike stationary monitors, this system allows for spatial mapping of air pollutants across different coordinates.
Core Sensors: Methane (CH_4) and Carbon Monoxide (CO) detection.
Connectivity: Real-time data transmission via Wi-Fi/LoRaWAN to a centralized dashboard.
Mobility: Integrated with a vehicular chassis to capture data points across a wide geographic area.
